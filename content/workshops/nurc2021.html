About this workshop

What it is: This is a hands on introduction aimed to beginners. The goal is to get participants familiar and enthusiastic with the R programming language. As well, to know how to access additional resources/courses to continue learning on their own.

Who is it for: This workshop is aimed towards researchers who are interested in using R but have little or no programming experience. This will be a gentle introduction to R and Jupyter.

Downloading the problem set

To download the problem sets, simply right click the .zip file and download/save it. If you want to take a look at the problems/content, you can check it out on GitHub.

Loading the problem set

To load the problem set, simply type jupyter lab or jupyter notebook in your terminal. Then, find the nurc_2021 folder and open then find the .ipynb file.

    (base) Andrews-MacBook-Pro:~ andrewli$ jupyter lab
